▄▀▄▀▄▀▄▀▄▀▄▀▄▀▄▀▄▀▄▀█▒░USING PGP░▒█▄▀▄▀▄▀▄▀▄▀▄▀▄▀▄▀▄▀▄

Sometime around January 7, 2014, Silk Road with be REQUIRING all addresses to be encrypted with PGP, many users are unsure how to do this, so I have written a brief step by step guide to installing, setting up, and using gpg4usb; a free PGP program that can be put onto a USB drive for easy access and use on any computer you use.

Installing gpg4usb:
1. Navigate to: http://gpg4usb.cpunk.de/ and download the latest version
2. Unzip the file into a folder on your computer or USB drive of your choice
3. Click either “Start_Windows” or “Start_Linux” depending on what operating system you're using.

Creating a private key
1. Open gpg4usb and click “Manage Keys” in the menu tab
2. In the window that opens up go to the “Key” menu and select “Generate Key”
3. Type in at least a name (which should obviously not be your actual name)
4. In the “Keysize” section, set it to 4096 or higher, anything lower will not be a secure bitsize for your key
5. Set the password for the key
6. Click “Okay” and the program will generate your key.

Importing another user's public key:
1. When a user gives you their public key through a PM or their profile on SR, copy the entire thing from "-----BEGIN PGP" to "KEY BLOCK-----"
2. In gpg4usb click the button labeled "import key" and select "clipboard" from the menu.
3. A dialogue box will appear with the user's key information, pay attention to the "name" field, as some users give their public key's silly names that are confusing.

Giving another user YOUR public key:
1. After creating a private key per the above instructions, open a new file in gpg4usb by selecting "File" and them "New" in the menu
2. Right click on your public key and select "Append selected key(s) to text"
3. Copy the information that appears in the file, this is your public key that you give to users

Decrypting a message you received from another user:
1. When a user sends you a message encrypted with your public key select the entire from the "-----BEGIN PGP" line to the "END PGP-----" and copy it
2. Open a new file in gpg4usb by selecting "File" and them "New" in the menu
3. Paste the entire message into the blank file.
4. Click the button labeled "decrypt"
5. Input the password you setup for your private key, and the contents of the file window will change to the message the user sent you.

Encrypting a message for another user:
1. After importing the public key of the user who you're sending the message to, go the the "Windows" menu in GPA and select "clipboard"
2. Type your message to the user in the window that opens up.
3. In the "File" menu select the "encrypt" option
4. Select the public key of the user you are sending the message to and click "OK"
5. A dialogue box informing you that you can't be certain this key belongs to that user will open up, press the "Yes" button
6. The text in the clipboard window will no longer contain the message you wrote, but an encrypted message that begins "-----BEGIN PGP MESSAGE-----" copy the entire contents of this window
7. Send the entirety of that text to the user who you are communicating with.
