Strong multi-factor authentication

The rapid growth of online identity theft has exposed the vulnerabilities of logging with a username and password. Any information that is static, or stored on a device connected to the Internet, can be copied and misused by malware.

Security experts agree that the best way to secure an online identity is to move out the user credentials to a separate device that is not connected to the Internet, and which generates a new and encrypted pass code every time it is used. Also, for strong two factor authentication it is recommended to combine the hardware authentication device with a PIN or a password – something you have + something you know.
YubiKey One Time Password

The Yubikey Standard is a hardware authentication device that changes passwords every time it is used. In it’s default mode, the YubiKey works by emitting an one time password (OTP) by emulating a USB keyboard. As USB keyboards are supported by almost all modern computers, the YubiKey requires no additional client software. (Windows, Linux, Mac, iPad, Firefox, Chrome, etc.)

OTP Usage

A typical implementation is shown in picture above – a user inputs a password (something you know), then inserts the YubiKey into the USB port and presses the button whereupon it will emit an OTP. A username is unnecessary as the OTP has a prefix which identifies the user. One step less, and one thing less to remember.

The OTP is verified by the software or service supporting the Yubikey, enabling secure login within a second. Each time a new OTP is generated by your YubiKey, it invalidates every other previously generated OTP. This means, even if your YubiKey’s OTP is ever recorded, it can never be used to access your YubiKey-protected site or software after it is used. Each OTP is unique to the YubiKey it’s generated from, and cannot be duplicated or copied to another YubiKey device. Your secure online identity can be held in your hand.
Supports almost any application

YubiKeys are available with different capabilities and form factors and for the range of enterprise, open source and consumer applications.

The YubiKey is supported by many password managers allowing users to secure and simplify the management of multiple passwords for legacy username/password systems.

While the YubiKey OTP does require a site or service to first implement the YubiKey Authentication server before it can be used, the YubiKey can also hold a traditional static password that works with any application supporting legacy username/password login. It can hold a password of up to 64 characters, allowing users to create exceedingly strong, complex passwords without having to remember them.
Durable, Secure & Affordable

The YubiKey is practically indestructible; moulded into one hermetically sealed mono-block it does not contain any moving parts or batteries. The exposed contacts are made of hardened gold, to be used for +100,000 inserts into a USB-port. YubiKeys have been immersed in salt water, run over by cars, and run through the wash and still work perfectly.

YubiKeys disrupt the existing relationship between usability, security and cost – offering great security with minimal cost and hassle.
